# Mushroom Classification

## Project Overview

This repository contains my solution for the **Mushroom Classification** machine learning competition completed as part of the IIT Madras BS Degree in Data Science.

The objective of the competition is to classify wild mushrooms as **Edible** or **Poisonous** based on their physical characteristics.

---

## Problem Statement

Build a binary classification model that predicts whether a mushroom is:

- Edible (e)
- Poisonous (p)

using the provided training dataset.

---

## Dataset

The dataset consists of various categorical features describing the physical properties of mushrooms, such as:

- Cap Shape
- Cap Color
- Gill Characteristics
- Stem Features
- Odor
- Habitat
- Other morphological attributes

Target Variable:

- **Class**
  - Edible (e)
  - Poisonous (p)
